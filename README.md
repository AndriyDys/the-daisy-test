# The Daisy Test

An interactive environmental web experience where every choice affects a living daisy.

## Files

- `index.html`
- `style.css`
- `script.js`
- `assets/`
- `flower-sprites/`

## Run locally

Open `index.html` in your browser.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload the project files to the repository root.
3. Go to `Settings -> Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and the `/ (root)` folder.
6. Save and wait for GitHub Pages to publish the site.

The site entry file is `index.html`, so it is already ready for GitHub Pages.
